La aplicación web consistirá en una cafetería online que te permitirá personalizar cafés, bebidas y comidas varias propias de una cafetería, ofreciendo una vista actual e intuitiva para las combinaciones de productos y precios.
La idea es que cada usuario registrado pueda personalizar sus productos y ponerlos a la venta, y cada vez que se haga una venta, se generen Loyalty Points que se podrán canjear dentro de la aplicación.
También se incluirá un sistema de chat, comentarios y likes para los productos visibles que cada usuario decida tener en su página, que servirá para dar feedback a los usuarios que publiquen y comunicarse con amigos también registrados en el sistema.
La idea principal de esta aplicación es crear una demo para ver diferentes productos que una cafetería podría ofrecer y valorar su popularidad y viabilidad en un entorno virtual y después implementarlo en la vida real.
No se llega a implementar el sistema de pago, sino que se hace un carrito de compra y una vez se confirme el pago se considera pagado.
