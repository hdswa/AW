Detalles

La aplicación web consistirá en una red social-cafetería online que te permitirá personalizar un menú de cafés y otras bebidas,
ofreciendo una vista actual e intuitiva para las combinaciones de productos y precios.

La aplicación tendrá 5 funcionalidades principales:

- Gestión de contenido:
En esta parte se hará una gran parte de las operaciones de la aplicación web: la personalización del menú, el sistema de login de usuarios,
la creación de una cafetería con un nombre único, la navegación por las cafeterías de otros usuarios y una barra de búsqueda para usuarios o cafeterías en concreto.

- Gestión de compra (Carrito incluido):
Cuando se entre a una cafetería, si se hace una compra, se añadirán los productos al carrito de compra del usuario.
El usuario podrá finalizar el pago en cualquier momento pero solo hará click, no se producirá ninguna transacción monetaria.

- Administrador (Página con todas las opciones):
Esto es una página solo accesible al administrador donde éste podrá: borrar usuarios, cafeterías, productos y comentarios,
además de realizar más operaciones desde la misma aplicación, sin necesidad de abrir la base de datos e introducir queries manualmente.

- Gestión de feedback (Comentarios, likes):
El usuario podrá dejar comentarios en cafeterías de otros usuarios. Sin embargo no podrá dejar comentarios en productos concretos, ya que consideramos que ésto
no sería muy eficiente y que es mejor ver un comentario en general que valore la cafetería en su conjunto.

- Gestión de mensajes (Chat):
Cada usuario tendrá una lista de de usuarios a los que sigue. Éstos podrán ser usuarios personales o cafeterías. Los usuarios podrán mandarse mensajes con otros usuarios o con otras cafeterias.
Otro punto a comentar es que en ningún momento los usuarios podrán ver el nombre del usuario propietario de una cafetería.
