Detalles

La aplicación web consistirá en una cafetería online que te permitirá personalizar cafés, bebidas y comidas varias propias de una cafetería,
ofreciendo una vista actual e intuitiva para las combinaciones de productos y precios.
La aplicación tendrá 5 funcionalidades principales

-Gestión de contenido
En esta parte se hará una gran parte de las operaciones de la aplicación web
como son la personalización de productos, el sistema de login de usuarios, creación de una cafetería con un nombre unico
la navegación por las cafeterías de otros usuarios, y una barra de busqueda para usuarios o cafeterías en concreto.

-Gestión de compra (Carrito incluido):
Cuando se entre a una cafetería, si se hace una compra, se añadirá los productos a un carrito de compra del usuario,
que el usuario podrá finalizar el pago en cualquier momento, pero solo se hace click en el pago pero no se produce transaccion de dinero.

-Administrador (Página con todas las opciones) :
Esto es una página solo accesible al administrador, donde podra: borrar usuarios, cafeterias, productos, Comentarios,
y más operaciones desde la misma aplicación sin necesidad de abrir la base de datos e introducir queries manualmente.

-Gestión de feedback (Comentarios, likes):
El usuario podrá dejar comentarios en cafeterias de otros usuarios pero no en cada producto ya que no muy eficiente ver los comentarios
de cada producto sino es mejor un comentario en general de la cafetería en conjunto

-Gestión de mensajes (Chat):
El usuario tendrá un sistema de amigos y un chat las cafeterías, podrá manderse mensajes entre los amigos, y podrá establecer
un chat directo con un usuario que tenga una cafetería, pero no sabrá el nombre de la persona sino solo el nombre de la cafeteria
